## 🎯 Problem Statement
Analyze e-commerce transaction data to drive data-driven decision making, improve customer targeting, and maximize revenue.

---

## 🚀 Key Features
- 📊 **Revenue Analysis** (Monthly trends & KPIs)
- 👤 **Customer Segmentation** using RFM Analysis
- 🏆 **Top & Low Performing Products**
- 🎯 **Rule-Based Recommendation System**
- 🌐 **Interactive Dashboard (Streamlit)**

---

## ⚙️ How It Works
1. Data is cleaned and transformed into a structured fact table  
2. RFM analysis is performed to segment customers  
3. SQL-style aggregations are implemented using Pandas  
4. Business insights are generated from customer and product behavior  
5. A rule-based recommendation system suggests products based on customer segments  
6. Results are visualized using an interactive Streamlit dashboard  

---

## 🧠 Business Insights
- Top customers contribute a significant portion of revenue  
- Revenue shows seasonal trends and fluctuations  
- Few products dominate sales performance  
- Many products underperform → optimization opportunity  

---
